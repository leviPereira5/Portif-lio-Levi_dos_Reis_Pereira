Sobre Esta Aplicação

1. Arquitetura MVT do Django

A aplicação foi desenvolvida utilizando a arquitetura MVT (Model View Template) do Django.

Componentes

  • Model: representa a base de dados
  • View: processa pedidos
  • Template: apresenta HTML

Fluxo de Informação

  1. O utilizador acede à página
  2. A URL chama uma View
  3. A View consulta os Models
  4. Os dados são enviados para o Template
  5. O HTML é apresentado ao utilizador
